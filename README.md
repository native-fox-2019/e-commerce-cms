# POST /users/register

## Register an account
## Endpoint
> /users/register

## Request body:
```
{
    email: string,
    password: string,
    role: string
}
```
All req.body is **required**

Role is between **Admin** and **User**

## Response
Status 201
```
{
    id: 1,
    email: "admin@admin.com",
    password: "hashedPassword",
    role: 'Admin',
    createdAt: 2020-03-03T15:01:27.405Z,
    updatedAt: 2020-03-03T15:01:27.405Z
}
```
Status 400
```
{
   status: 400,
   msg: error message
}
```
Status 500
```
{
   status: 500,
   msg: "Internal Server Error"
}
```
